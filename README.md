Rock Paper Scissors – Python

A simple console-based Rock Paper Scissors game developed in Python, focusing on clean logic and core programming fundamentals.

Overview

The application allows the user to select rock, paper, or scissors, while the computer generates a random choice. The program evaluates both inputs and determines the result according to standard game rules.

Skills & Technologies

Python

Conditional logic (if / elif / else)

Lists and dictionaries

Random module

User input handling

Basic input validation

Clean and readable code structure

Purpose

This project was created as part of a Python bootcamp to strengthen problem-solving skills, logical thinking, and familiarity with fundamental programming concepts used in real-world applications.
